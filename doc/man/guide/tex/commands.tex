\chapter{かな戳机恃垂ユ〖ティリティ}
\label{4かな戳机恃垂ユ〖ティリティ}

塑システムではかな戳机恃垂辑今のメンテナンスやかな戳机恃垂サ〖バのメン
テナンスを乖うための硷」のユ〖ティリティを捏丁しています。ユ〖ティリティ
は辑今をメンテナンスするものとその戮の2つに络きく尸けられ、辑今をメン
テナンスするものに簇しては、かな戳机恃垂サ〖バを拆して辑今のメンテナン
スが材墙なものと、辑今ファイルが赂哼するマシン惧でなければメンテナンス
ができないものがあります。

この鞠ではこれらのユ〖ティリティツ〖ルに簇する棱汤をマニュアルペ〖ジの
妨で绩します。

辑今の恃垂や、糠しい辑今の侯喇などのために笆布のようなユ〖ティリティを
捏丁しています。

\begin{nquote}{2em}
\begin{namelist}{cannaserverxxxx}
\item [mkromdic] テキスト妨及のロ〖マ机かな恃垂テ〖ブルをバイナリ妨及に恃垂する。
\item [mkbindic] テキスト妨及の辑今をバイナリ妨及の辑今に恃垂する。
\item [dpbindic] バイナリ妨及の辑今をテキスト妨及の辑今に恃垂する。
\item [dpromdic] バイナリ妨及のロ〖マ机かな恃垂テ〖ブルをテキスト妨及に恃垂する。
\item [dicar] バイナリ妨及の辑今のア〖カイブを乖う。
\end{namelist}
\end{nquote}

笆布はかな戳机恃垂サ〖バを拆して辑今のメンテナンスが材墙なツ〖ルです。

\begin{nquote}{2em}
\begin{namelist}{cannaserverxxxx}
\item [mkdic] 辑今の侯喇】アップロ〖ドを乖う。
\item [lsdic] 辑今の办枉を叫蜗する。
\item [rmdic] 辑今を猴近する。
\item [mvdic] 辑今の叹涟を恃构する。
\item [cpdic] 辑今をコピ〖する。
\item [catdic] 辑今のダウンロ〖ドを乖う。
\item [addwords] 帽胳の办崇判峡を乖う。
\item [delwords] 帽胳の办崇猴近を乖う。
\item [chmoddic] 辑今のアクセス涪の恃构を乖う。
\end{namelist}
\end{nquote}

また、介袋觉轮のチェックや、かな戳机恃垂サ〖バに簇する笆布のようなコマ
ンドを捏丁しています。

\begin{nquote}{2em}
\begin{namelist}{cannaserverxxxx}
\item [cshost] かな戳机恃垂システムのサ〖バˇアクセス扩告觉轮を山绩する。
\item [cannacheck] 泣塑胳掐蜗システム簇息攫鼠を山绩する。
\item [cannakill] かな戳机恃垂サ〖バcannaserverを姜位させる。
\item [cannaserver] かな戳机恃垂のサ〖ビスを捏丁する。
\item [cannastat] かな戳机恃垂サ〖バの攫鼠を山绩する。
\end{namelist}
\end{nquote}
